# CAUdigital

CAU Digital es una aplicación web que permite consultar rápidamente el estado del servicio y acceder a herramientas útiles para planificar tu viaje.

La página muestra información actualizada del servicio y ofrece distintas acciones para que puedas resolver lo que necesitás sin perder tiempo.

¿Qué podés hacer?

En CAU Digital podés:

-Ver el estado actual del servicio

-Planificar un viaje

-Consultar horarios

-Acceder a más información del servicio

¿Cómo usar la aplicación?

1. Entrá a la página.
2. Mirá el estado del servicio que aparece en pantalla.
3. Elegí la acción que necesitás.

Las opciones disponibles son:

Ver más → información ampliada del servicio
Planificar viaje → herramientas para organizar tu recorrido
Horarios → consulta de horarios de trenes
Otros servicios → accesos adicionales relacionados con el sistema

Acceso:
La aplicación funciona directamente desde el navegador y está optimizada para usarla desde el celular.
